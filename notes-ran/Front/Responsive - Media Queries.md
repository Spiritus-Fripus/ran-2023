```css

@media (max-width:800px) {
	h1 {

	}
}
```

**Média Queries**
On utilise la balise **@media** avec une taille  _(max-width:800px)_ 
Les propriétés s'appliquerons quand l'écran passe en dessous de 800px

_exemple taille écran_:

xs **<** 576px
sm **>=** 576px
md **>=** 768px
lg **>=** 992px
xl **>=** 1200px
xxl **>=** 1400px
